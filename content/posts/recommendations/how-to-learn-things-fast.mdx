# Summary

當想學習新體系時，先了解它的框架或是整體脈絡，效果會好過只是鑽研在特定知識點上

1. 參考別人已有的框架：先看整體脈絡，了解各個知識點的關係，如此在深入知識細節時才有著明確目的
   1. 參考書書目
   2. chatGPT 快速了解
   3. 網路上別人統整的快速入門
   4. 只要是能讓你快速吸收的東西都可以，不管多奇特或多單薄
2. 使用 timeline 梳理脈絡：理解體系的歷史
   1. 從頭開始理解比較容易
   2. 基於結果導向型數據的 timeline，歷史與數據是有關聯的
   3. 盡可能找到第一手資料
3. 從流程下手：羅列完成一件事情需要經歷的步驟，這樣才能知道自己缺少哪些知識點
4. 其他技巧
   1. 查看論文或長文讀兩遍，第一遍只看概念，第二遍再看細節
   2. 進行知識輸出
