# Summary
要預估一件事多久會完成有三個面相：複雜度、時間、錯估的風險，降低風險最有效的方式就是工作拆解，拆得越細風險越低

# Main idea
提出預估事情的元素有
1. 誰該做
2. 何時要做完
3. 要做多久
4. 誰是低效者

再將 3 獨立出來講解，並提出可能的解法
