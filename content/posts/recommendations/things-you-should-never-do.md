# Summary
為何大部分的 programer 都喜歡重寫 code? 因為讀 code 遠比寫 code 難。
不要隨意重寫 code，因為你等於是丟掉過去所有人的知識之後再寫自己認為更好的—其實只是重頭踩一遍所有的問題，或是達成一件已經毫無經濟效益的事

重構不是重寫，是每次的修改都會讓程式更加好懂
