python 3 原生提供了一個簡易的 cache decorator 叫 [lru_cache](https://docs.python.org/3/library/functools.html#functools.lru_cache)，可以把耗時的函數執行結果儲存起來，一段時間就刪掉

舉例來說下面這段 code
```py
@lru_cache()
def say_hi(name: str):
    return f'Hello {name}'
```
