我用的是 macbook pro 2019 rentina 13.3，對應的型號是 A1989

# 需要道具
<TableWrapper className="table table-bordered">
|中文|英文|尺寸|用途|
|:--:|:--:|:--:|:--|
|梅花螺絲刀 T3|torx screwdriver T3|1.27mm|拆連接觸控板 2 顆螺絲|

# 卸除電池步驟
1. 確認電池型號
<Figure src="https://lh3.googleusercontent.com/9Z9MBl-iUDSv2F_ZgORCzcggNL6W35pd2DQYrepKKNnysWjrC9vq61jQy_JvXShc8zAtBZF22VE9842SICdGG1WPEOYPdH7Xy-uV15s" caption="底版都會標注電池型號" />

2. 拆解背板
<Figure src="https://i.ebayimg.com/images/g/zDQAAOSwvaJgFzIG/s-l1600.jpg" caption="使用 P5 進行拆解" />
   1. 背板中央是用卡的，要稍微用力往上掰一下
   2. 背板與螢幕連結處（凹陷處）是內嵌入的，要往外推

3. 移除電池充電線：這步驟要特別小心，不要把線弄壞了導致無法正常充電，線的兩端是用卡榫卡住，可以用指甲輕輕摳起方可移除

4. 移除觸控板
<Figure src="https://i.imgur.com/TVbQV8Jl.jpg" caption="紅圈處使用 T3 移除，上方黑色那條是連接觸控板的線" />
   1. 使用 T3 將連接觸控板的螺絲釘紐掉
   2. 使用指甲把排線拔除
   3. 緩步撕掉膠條
   4. 抬起螢幕進行移除


5. 移除電池
<Figure src="https://i.imgur.com/atZnOox.jpg" caption="特別小心不要把電池戳壞了，不然會燒起來" />
   1. 使用 T5 移除中間兩側螺絲
   2. 使用粉底刮刀刮開電池與面板間的黏膠
   3. 全部的黏膠都清掉後即可用手拔開
<Figure src="https://i.imgur.com/KLJNQXe.jpg" caption="成功拆除後" />

# 安裝電池步驟
1. 安裝電池：macbook 的電池是用黏的，把新電池的膠膜拆下，對好位置後黏回面板
   1. 注意充電面板不要折到
   2. 使用 T5 鎖緊電池中間兩側螺絲
2. 裝回觸控板（先不要鎖螺絲）
3. 裝回電池充電線
4. 開機測試觸控板與充電功能，都沒問題再把觸控板螺絲鎖上
